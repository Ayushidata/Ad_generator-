This project is a multimodal AI system that generates complete advertisement packages — including scripts, audio ads, and poster designs — using Large Language Models, Text-to-Image generation, and Text-to-Speech. The backend runs on FastAPI and the frontend uses Streamlit for a clean and interactive UI.

Features
A detailed list of everything the project can do.

Generate 3 creative ad scripts using TinyLlama

Automatically produce 4K ad posters using Stable Diffusion v1.5

Convert scripts into audio ads (MP3) using gTTS

Evaluate each ad's effectiveness and relevance

Generate critique & improvement suggestions

Streamlit-based user interface

FastAPI backend for API integration

CUDA support for performance (if available)
